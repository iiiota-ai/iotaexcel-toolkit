# Changelog

## 1.0.0

- Initial IotaExcel ToolKit release.
- Added Convert and Codegen command palette workflows.
- Added Excel conversion to `.bytes`, JSON, and CSV.
- Added C# reader code generation.
- Added readonly `.bytes` preview with search, row jump, column jump, pagination, and resizable columns.
- Added separate Convert and Codegen config/input/output settings.
- Added bundled cross-platform IotaExcel executables.
